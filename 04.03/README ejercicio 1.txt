Archivo XML que representa un catálogo de software informático, dividido en dos categorías usando espacios de nombres:

- os: Para software de sistemas operativos.
- office: Para software de aplicaciones de oficina.

Cada software tiene los siguientes elementos:
- nombre: Nombre del software.
- version: Versión actual.
- desarrollador: Empresa o persona que lo desarrolló.
- url: Enlace a más información sobre el software.
